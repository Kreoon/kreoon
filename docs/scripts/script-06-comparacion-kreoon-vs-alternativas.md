## NOTAS DE PRODUCCION

### Motion Graphics necesarios
- Logos de cada herramienta (Notion, Trello, Asana, ClickUp, Google Sheets, WhatsApp, Kreoon) — PNG transparente
- Tabla comparativa animada — se llena en tiempo real durante el video
- Checks y X animados para la tabla
- Veredicto final de cada herramienta en card visual

### Tono critico
- No atacar a las herramientas — posicionarse como educador, no como competidor agresivo
- Cada herramienta tiene su lugar — la honestidad genera mas confianza que el ataque
- El verdadero competidor es "no usar nada" / WhatsApp — ahi es donde hay mas audiencia
