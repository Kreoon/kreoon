## NOTAS DE PRODUCCION

### B-Roll necesario
- Pantalla de WhatsApp con mensajes acumulados — **Stock / grabacion propia** / Static / 3s
- Dashboard principal de Kreoon — **Screen recording** / Zoom in suave / 8s
- Workspace o escritorio real — **Selfie cam / B-roll propio** / Tracking / 4s
- Manos en teclado trabajando en Kreoon — **Grabacion propia** / Static / 3s

### Musica sugerida
- Genero: Lo-fi instrumental / hip-hop suave
- Dinamica: Empieza bajo en hook (tension), sube en promesa, mantiene en contenido
- Referencia: "Tokyo Lo-Fi" o similar sin copyright
