## NOTAS DE PRODUCCION

### B-Roll necesario para cada tutorial
- Navegacion inicial hacia el feature — **Screen recording** / Zoom progresivo / 15s
- Demo de configuracion completa — **Screen recording** / Tiempo real + cortes / 180s
- Resultado final visible en pantalla — **Screen recording** / Zoom in / 10s
- Talking head de introduccion y tips — **Camara principal** / Static / variable

### Configuracion de Screen Recording
- Resolucion: 1920x1080 minimo
- Cursor: resaltado con halo de color (OBS o Screenflow)
- Zoom: usar zoom in/out para guiar la atencion (shortcut en Screenflow o similar)
- Velocidad: grabar en tiempo real, cortar silencios en edicion

### Anotaciones en post-produccion
- Circulos o flechas rojas para llamar atencion
- Texto descriptivo en momentos de navegacion
- Badge "RECOMENDADO" / "ERROR" / "TIP" con colores consistentes
