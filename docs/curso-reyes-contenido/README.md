# Los Reyes del Contenido - Curso de Kreoon

## Visión General

Curso completo de la plataforma Kreoon para creadores de contenido, editores y agencias en LATAM. El curso alimenta la comunidad "Los Reyes del Contenido" en Skool y genera contenido para YouTube, TikTok e Instagram.

**Instructor**: Alexander Cast
**Plataforma**: Skool
**Modelo**: Free + Premium ($39/mes)
